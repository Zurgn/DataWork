# Вариант 12
**Используемые данные:** *Физические запасы воды* в Новой Зеландии в период с 1995 по 2020 год

*Физические запасы воды* — это количественное измерение объема воды, которая находится в распоряжении региона в конкретный момент времени

| Название колонки | Описание |
| :--- | :--- |
| **region** | Регион |
| **variable** | Событие |
| **RID** | ID региона |
| **yq** | Год, квартал |
| **value** | Объем воды |
| **year** | Год |
| **Series** | Тип данных |
| **Unit** | Единица измерения (миллион кубических метров) |
| **Source** | Источник данных |
| **Quarter** | Квартал |
